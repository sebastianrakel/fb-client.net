fb-client.NET

a filebin client for Windows, written in C#.NET

About

filebin is a pastebin like self-hosting service, written by bluewind.

Features

  • paste plain text
  • paste files
  • show history
  • drag & drop file
  • context menu integration
  • systray
  • screenshot

Upcomming

  • paste multiple files
  • compression
  • auto update

Requirements

  • Microsoft .NET 3.5
  • a filebin account
